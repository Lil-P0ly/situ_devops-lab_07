# Лаб 7. Kuber, rolling update

## Куберизировать приложение flask+redis

создать манифесты для запуска приложения flask и БД redis в кластере, и сервисы сетевого доступа к ним.

### Развертывание:
```bash
kubectl apply -f flask_k8s/
```
